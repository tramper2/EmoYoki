User Flow (사용자 흐름)
3.1 공급자 (이모님)
가입/로그인: 휴대폰 번호 인증 및 프로필 특히 근무자 얼굴 사진, 입금받을 계좌 등록. 범죄 예방을 위해 본인 인증이 되어야 한다. 
근로계약서 형태등 법적으로 필요한 동의서에 동의를 받는 과정을 진행한다.

활동 구역 설정: 지도에서 중심점 지정 + 반경 선택(2km/5km/10km).
대기/수락: 설정 구역 내 호출 발생 시 푸시 알림 확인 → 상세 내용 확인 후 수락.
수행: 현장 도착 버튼 → 업무 완료 사진 업로드 → 종료.

3.2 구인자 (고용자)
역시 계약서 형태의 동의서등에 동의를 받는 과정을 진행한다.
구인자 역시 구인자로 등록이 필요하며 미리 입금이 된 후에 호출이 가능하다.
범죄등의 작업에 호출되어서는 안되기 때문에 좀 더 엄격한 신원확인이 필요하다.
역시 본인 인증을 위해 전화번호와 통장 일치를 통한 방법등으로 신원확인을 한다.

메인: 필요한 도움 카테고리 선택.
호출: 지도에 근무할 위치표시 및 문자로 건물이름 등 장소, 시간, 요청사항 입력 후 호출.
매칭: 수락한 이모님 프로필 확인 → 채팅/전화 연결.
평가: 서비스 완료 확인 및 별점 부여.
