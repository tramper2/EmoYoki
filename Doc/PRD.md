서비스명: 이모~ 여기! (Emo-Here)
나노바나나를 이용해 어울리는 그림을 그려 배치한다
슬로건: "우리 동네 믿음직한 손길이 필요할 때, 이모~ 여기!"

이번 개발은 본 개발이 아니라 테스트 용도로 구인자 id한개 공급자 id 2개를 메인홈페이지에 클릭하여 로그인한걸로 처리한다
구인자는 구인 공고를 게시할 수있고 공급자는 구인 게시물에 지원할 수 있다
메인페이지는 더미 구인 공고 데이터를 몇개 생성해서 실제 서비스에서 어떻게 보일수 있을지 짐작할 수 있게 해준다


1.1 핵심 가치
신뢰: 중장년층의 연륜을 활용한 생활 밀착형 서비스.
자율성: 이모님이 직접 활동 중심지와 반경(2/5/10km)을 설정.
편의성: 30분 단위의 초단기 업무 및 직관적인 UI.

1.2 주요 기능
호출 서비스: 병원 동행, 강아지 산책, 가사 보조, 가게 땜빵 등.
위치 기반 매칭: 근로장소와 이모님이 설정한 활동 반경 내 호출 발생 시 실시간 알림.
업무 보고: 사진 첨부 기반의 완료 보고서 자동 생성.
신뢰 지수: '동네 온도' (평점 및 매칭 횟수 기반).
